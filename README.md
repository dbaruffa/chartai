# ChartAI

Prompt engineering project to have an AI that helps you create charts.

This project is based heavily on this project: https://github.com/safak/chatgpt-clone/tree/completed<br>
See this accompanying video: https://www.youtube.com/watch?v=8iAQ1h30n5I

## Setup & Run

1. Install npm
2. Clone this repo
3. cd to repo
4. `npm install`
5. `npm run`
6. Open browser and browse to displayed address (e.g. http://localhost:5173/)
